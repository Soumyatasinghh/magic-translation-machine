# 📄 Magic Translation Machine

An AI-powered web application that translates PDF and DOCX documents into multiple languages while generating translated text, audio narration, and an animated video from the translated content.

Designed with a modern and responsive interface, the application combines document processing, machine translation, text-to-speech synthesis, and multimedia generation into a single workflow.

---

## 🚀 Features
- 📄 Upload PDF and DOCX documents
- 🌍 Translate documents into:
English
Hindi
Bengali
- 🔍 Automatic text extraction from uploaded files
- 🔊 Text-to-Speech (TTS) audio generation
- 🎥 Animated video generation with synchronized narration
- 🎨 Modern Glassmorphism user interface
- 📥 Download translated audio
- 📥 Download generated video
- ⚡ Fast and lightweight document processing

```bash

```

```bash
  git clone https://github.com/your-username/Magic-Translation-Machine.git
```
- Install the required dependencies:
```bash
pip install -r requirements.txt
```
- Run the application:
```bash
python app.py
```
- Open your browser and visit:
```bash
http://127.0.0.1:5000
```

## 📖 How It Works
- Upload a PDF or DOCX document.
- Extract text from the uploaded document.
- Translate the extracted content into the selected language.
- Generate translated audio narration.
- Create an animated video synchronized with the generated audio.
- Display the translated text, audio player, and video.
- Download the generated audio and video.
