📄 Magic Translation Machine

An AI-powered web application that translates PDF and DOCX documents into multiple languages while generating translated text, audio narration, and an animated video from the translated content.

Designed with a modern, responsive interface, the application combines document processing, machine translation, text-to-speech synthesis, and multimedia generation into a single workflow.

🚀 Features
📄 Upload PDF and DOCX documents
🌍 Translate documents into:
English
Hindi
Bengali
🔍 Automatic text extraction
🔊 Text-to-Speech (TTS) audio generation
🎥 Animated video generation with synchronized narration
🎨 Modern glassmorphism UI
📥 Download translated audio and generated video
⚡ Fast and lightweight processing

⚙️ Installation
Clone the repository:git clone https://github.com/your-username/Magic-Translation-Machine.git
Navigate to the project directory: cd Magic-Translation-Machine
Install the required dependencies:pip install -r requirements.txt
Run the application: python app.py
Open your browser and visit: http://127.0.0.1:5000

📖 How It Works
Upload a PDF or DOCX file.
The application extracts the document text.
The extracted content is translated into the selected language.
A translated audio narration is generated.
An animated video is created using the translated content and synchronized audio.
Users can view and download the generated audio and video.
